# Simple Python Addition Script

This is a basic Python script that demonstrates how to create and use a simple function.

## Description
The script defines a function called `add_numbers` which takes two numbers as input and returns their sum.

## How it works
- Two numbers are assigned to variables
- The function is called with those numbers
- The result is printed to the console

## How to run
Make sure Python is installed, then run:

python add.py
